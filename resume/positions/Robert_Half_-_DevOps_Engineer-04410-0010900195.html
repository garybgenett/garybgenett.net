      DevOps Engineer

      <Back Next>
      Have an account? Sign In

      Location:

      Seattle, WA

      Date Posted:

      March 18, 2019

      Employment Type:

      Permanent

      Job Reference:

      04410-0010900195

      Staffing Area:

      Technology & IT

      Description
      • Education/Experience – B.S. in Computer Science, Computer Engineering, or Software Engineering with 5 years’ work experience.
      • Databases – Experience installing, administering, tuning, monitoring and supporting databases (Oracle, PL/SQL, PostgreSQL, MySQL).
      • Systems – Experience installing, configuring, maintaining, monitoring and administering operating systems (Windows, CentOS, RHEL, OEL).
      • Virtualization – Experience with virtualization technologies, tools and practices (VMware ESXi/vSphere, Citrix XenApp/Studio).
      • Automation – Experience developing deployment and configuration management automation using open-source software (Ansible, Jenkins, TeamCity, Terraform, Chef, Puppet).
      • Scripting – Experience writing robust, reliable code with platform-specific scripting languages (Bash, PowerShell, Groovy).
      • Networking – Experience with general networking (TCP/IP, DNS, DHCP, Firewalls) and administering Windows domains (Active Directory, Group Policy).
      • SCM – Experience with SCM tools (Git/Bitbucket, Perforce, Subversion) and source code management and integrations.
      • Cloud – Working knowledge of Cloud deployment, operation and administration (Amazon AWS, EC2, S3).
      • DevOps – Working knowledge of additional DevOps tools (Docker, Kubernetes, Kafka, ActiveMQ, Consul, Prometheus) and practices (Continuous Integration/Continuous Deployment)
      Requirements
      Linux, Windows Azure, Microsoft Windows, Ansible, Puppet, Chef, Python, Perl
